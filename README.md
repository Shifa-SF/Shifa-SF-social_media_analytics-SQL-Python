# Social Media Analytics using SQL & Python

## 📌 Project Overview
This project analyzes social media user activity and engagement using MySQL and Python.

## 🛠 Technologies Used
- MySQL
- Python
- Pandas
- Matplotlib

## 📊 Features
- Designed relational database (Users, Posts, Likes, Comments)
- Performed SQL joins and aggregation queries
- Calculated user activity and engagement metrics
- Visualized data using bar charts
- Exported results to CSV

## ▶ How to Run
1. Create the MySQL database and tables.
2. Insert sample data.
3. Update MySQL credentials in the Python file.
4. Run:
   python social_media_analysis.py

## 📈 Sample Output
- Most Active Users Graph
- Engagement Analysis
